# 법인 ETF 100세 시뮬레이터 — 유앤김 자산관리

> 1966년생(만60세, 2026년) → 만100세(2066년) | SCHD ₩10억 + TLTW ₩4억 | 법인 투자 시뮬레이터

## ⚠️ 검증 결과 반영 (v1.1 수정사항)

외부 코드 검증을 통해 발견된 6개 문제를 모두 수정했습니다.

| # | 문제 | 수정 내용 |
|---|------|----------|
| 1 | 수동입력 버튼 함수 충돌 | app.js의 중복 `openManualInput()` 제거. manual-input.js의 `manualModal` 단일화 |
| 2 | 40년 필터 인자 누락 오류 | HTML `onchange`를 `renderSimTableFiltered()`로 수정 (rows 자동 전달) |
| 3 | 첫해 배당 계산 기준 모호 | "기초자산(원금) 기준"으로 명확화. 만60세 첫해는 투자원금 그대로에 배당률 적용 |
| 4 | 만100세 연도 불일치 (2065 vs 2066) | 2026+40=**2066년**으로 전체 통일 |
| 5 | "동기화" 표현 과장 | "이 기기 저장"으로 명칭 변경. 실제 기기 간 이전은 **내보내기(JSON)/가져오기** 버튼으로만 가능 (자동 클라우드 동기화 아님) |
| 6 | 세무 반영 부족 | 요약 화면 상단에 "반영됨/미반영" 세무 항목을 명시한 고지 배너 추가 |

## 현재 세무 반영 범위 (정확한 고지)

**반영됨**: 미국 배당 원천징수세 15%만 반영

**미반영** (별도 세무사 확인 필수): 법인세, 가수금 인정이자 과세, 배당소득 종합과세, 비상장주식 보충적평가법, 증여세/상속세, 환율 변동, ETF 실제 분배금 변동

## 기능
- **요약 대시보드** — 세무 고지 배너 + KPI 카드 + 차트(3종) + 이정표
- **가정 설정** — 모든 투자 파라미터 실시간 수정 및 재계산
- **40년 시뮬레이션** — 연도별(만60~100세) 자산·배당·가수금 테이블, 정상 작동하는 구간 필터
- **월현금흐름** — 세후 배당으로 가수금 우선 충당 구조
- **자녀지분가치** — 80/85/90/95/100세 기준 주주별 지분 시각화
- **수동 입력** (manual-input.js 단일 모듈) — 실제 배당수령액, 가수금 상환액, 주가 기록
- **이 기기 저장** — localStorage 기반 (자동 클라우드 동기화 아님)
- **내보내기/가져오기** — JSON 파일로 실제 기기 간 데이터 이전 가능

## 데이터 이전 방법 (기기 변경 시)
1. 기존 기기: 헤더의 ⬇ "백업 파일 내보내기" 클릭 → JSON 파일 다운로드
2. 새 기기: 헤더의 ⬆ "백업 파일 가져오기" 클릭 → 저장된 JSON 파일 선택
3. 자동으로 가정값·실제입력기록·메모가 복원됨

## GitHub Pages 배포 방법
1. `yjjn2005` 계정에 `corp-etf-sim` 레포지토리 생성
2. 이 폴더 내용을 Push
3. Settings → Pages → Source: GitHub Actions 또는 main 브랜치 선택
4. 배포 확인

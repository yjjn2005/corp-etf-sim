# 법인 ETF 100세 시뮬레이터 — 유앤김 자산관리

> 1966년생(만60세, 2026년) → 만100세(2066년) | **SCHD ₩10억 + JEPQ ₩2.5억 + JEPI ₩1.5억** | 법인 투자 시뮬레이터

## 배포 URL
```

```

## 포트폴리오 구성 (v4)

| 종목 | 투자원금 | 특성 | 배당률(초기) | 주가성장률 |
|------|---------|------|------------|---------|
| SCHD | ₩10억 | 고배당 성장형 (Schwab) | 3.3% → 성장 | 5%/년 |
| JEPQ | ₩2.5억 | 나스닥 프리미엄 인컴 (JPMorgan) | 10% | 3%/년 |
| JEPI | ₩1.5억 | S&P500 프리미엄 인컴 (JPMorgan) | 8% | 2%/년 |

> **v3 → v4 변경**: TLTW(장기채 커버드콜, 금리 위험) → JEPQ+JEPI(주식형 커버드콜, 원금 성장 반영)

## 계산 구조

- **시뮬레이션 기간**: 만60세(2026) ~ 만100세(2066) = **41개년** (60,61,...,100세)
- **배당 기준**: 기초자산(각 연도 시작 시점 자산) 기준으로 배당 산정
- **가수금 상환**: 세후 배당 → 가수금 우선 상환(**연 4,800만원 한도**) → 잉여금 적립
- **세금 반영**: 미국 배당 원천징수세 15%만 반영
- **미반영 세금**: 법인세, 가수금 인정이자, 배당소득 종합과세, 증여세/상속세, 환율 변동 (세무사 확인 필요)

## 주요 기능

- **요약 대시보드** — KPI 6종(월배당×4 + 누계배당 + 가수금완납시점) + 자산 막대 + 배당 추이
- **가정 설정** — SCHD/JEPQ/JEPI 파라미터 실시간 수정
- **연도별 시뮬레이션** — 만60~100세 41개년 테이블, 구간 필터
- **월 현금흐름** — 3종목 개별 월배당 + 상환/잉여 흐름
- **자녀 지분가치** — 80~100세 기준 주주별 지분 시각화
- **수동 입력** — 실제 SCHD/JEPQ/JEPI 배당수령액, 현재가 기록
- **이 기기 저장** — localStorage (기기 간 이전은 내보내기/가져오기 JSON 활용)
